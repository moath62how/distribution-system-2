اختبار صفحة تفاصيل العميل

اختر عميل لاختبار صفحة التفاصيل:

عميل رقم 1 - اختبار تفاصيل العميل عميل رقم 2 - اختبار تفاصيل العميل العودة لصفحة العملاء الرئيسية

الميزات المحدثة:

  • تحسين معالجة الأخطاء في المدفوعات والتسويات
  • مسح النماذج تلقائياً بعد الإرسال الناجح
  • عرض الرصيد بالألوان الصحيحة (أحمر = له علينا، أخضر = عليه لنا)
  • تحسين رسائل الخطأ والنجاح
  • تعيين التاريخ الافتراضي لليوم الحالي
  • مسح الرسائل عند فتح وإغلاق النوافذ المنبثقة
